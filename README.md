# Intelligent Resume Analyzer

## Project Description
The Intelligent Resume Analyzer is a Python-based application that automates resume screening. 
It parses resume text, extracts important candidate details, compares them with job requirements, 
and generates a match score along with a hiring recommendation.

This tool helps recruiters quickly evaluate candidates based on their skills and experience.

---

## Features
- Extracts candidate information (Name, Email)
- Identifies skills from resume text
- Calculates match score (0–100)
- Generates hiring recommendation
- Saves analysis results in JSON format
- Produces a clear resume analysis report

---

## Technologies Used
- Python
- Regular Expressions (re)
- JSON

1. Clone the repository
